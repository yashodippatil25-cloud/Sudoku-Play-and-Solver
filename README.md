# 🧩 SudokuMaster
-<< Sudoku Play & Solver >>-

A responsive Sudoku web application where users can either enjoy solving Sudoku puzzles or instantly solve their own puzzles using a C++ backtracking algorithm.

This project was built to strengthen my understanding of web development, database integration, and algorithm implementation.

---

## ✨ Features

### 🎮 Play Mode
- Three difficulty levels (Beginner, Normal, Pro)
- Real-time timer
- Mistake counter
- Reset and New Game options
- Multiple UI themes

### ⚡ Solve Mode
- Enter any valid Sudoku puzzle
- Instant solution using a C++ Backtracking Solver
- Detects invalid or unsolvable puzzles

## ⚙️ Getting Started

1. Clone this repository

```bash
git clone https://github.com/yashodippatil25-cloud/Sudoku-Play-and-Solver.git
```

2. Import the database

```
database/sudoku_db.sql
```

3. Update your database configuration in

```
config/db.php
```

4. Run the project using XAMPP or Laragon.
